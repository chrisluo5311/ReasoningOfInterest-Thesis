# Santa Clara University Ph.D. Thesis Template
## Original Website
https://www.scu.edu/engineering/current-student-resources/current-phd-students/department-resources/

## Author
Cihan Ruan


This is a LaTeX template transferred from word version of Santa Clara University PhD dissertations.

## Features
- Properly formatted title page
- Dedication page
- Acknowledgments page
- Abstract page
- Proper spacing and font sizes according to SCU requirements
- Times New Roman font
- 1.5 line spacing for main content

## Usage
1. Fill in your information in the designated fields
2. Compile with pdfLaTeX
3. Follow SCU thesis guidelines for content
